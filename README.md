# dataBase_ecommerce
Este projeto visa a modelagem e implementação de um sistema de banco de dados relacional (Minimundo) para uma plataforma de comércio eletrônico (e-commerce) fictícia chamada "Lili&amp;Nana". O foco é projetar a estrutura de dados, garantindo a integridade transacional e a automação de processos através do uso estratégico dos recursos do MySQL.
